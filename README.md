# 📚 Projetos em VisualG

Este repositório reúne projetos desenvolvidos durante meus estudos de **Lógica de Programação** utilizando **VisualG (Portugol)**. O objetivo é consolidar conceitos fundamentais que servirão como base para o desenvolvimento em linguagens como JavaScript, Java, C# e outras.

## 🎯 Objetivo

Desenvolver a capacidade de resolver problemas através de algoritmos, implementando sistemas completos que simulam situações do mundo real.

## 📂 Projetos

### 🏦 Sistema Bancário

Simulação de um sistema bancário com operações essenciais para gerenciamento de contas.

**Principais funcionalidades:**

- Cadastro de contas
- Login com CPF e senha
- Depósito
- Saque
- Transferência entre contas
- Consulta de saldo
- Alteração de senha
- Exclusão de conta
- Validação de CPF
- Controle de usuários conectados

---

### 📚 Sistema de Biblioteca

Simulação de um sistema de gerenciamento de biblioteca.

**Principais funcionalidades:**

- Cadastro de livros
- Cadastro de usuários
- Empréstimo de livros
- Devolução de livros
- Pesquisa por título
- Pesquisa por autor
- Validação de CPF
- Validação de CEP
- Controle de disponibilidade dos livros

Durante o desenvolvimento destes projetos foram praticados conceitos fundamentais para o desenvolvimento de software, como organização do código, resolução de problemas, modelagem de sistemas e implementação de regras de negócio utilizando apenas recursos da linguagem.
